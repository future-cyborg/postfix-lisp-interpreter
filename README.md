# postfix-lisp-interpreter
Interpreter for a post-fix style basic Lisp

> ((2 3 *) ((5 7 -) 1 +) *)


#### Installation
Use make to compile with g++ on linux machine.
```
make
```
#### Usage
Call postlisp from command line with -i flag for repl mode. -h flag for help.
```
./postlisp -i
```
